# HR Analytics Dashboard

## Project Overview

The HR Analytics Dashboard is an interactive business intelligence project developed using Power BI, SQL, and Excel. This dashboard helps analyze employee data and provides meaningful insights related to workforce management, employee attrition, salary distribution, hiring trends, and department performance.

## Tools & Technologies Used

* Microsoft Power BI
* MySQL
* Microsoft Excel

## Key Features

* Total Employees KPI
* Average Salary Analysis
* Attrition Percentage Tracking
* Department-wise Employee Distribution
* Gender Distribution Analysis
* Employee Joining Year Trends
* Top Highest Paid Employees
* Interactive Visualizations and Filters
